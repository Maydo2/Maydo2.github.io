Project: Zombie Game



During this project, I had to challenge myself to learn a lot of new things. The purpose of this project was to create a zombies game where there would spawn endless enemies from multiple points on the map. In order to make this work I had to learn the following skills:
  • Raycasting
  • Spawn Points
  • Creating Objects
  • HP System
  • Stamina System
